Three components of metaseq

  1. Genomic data accessed by interval
  2. Work with tables of processed results, keyed by gene ID
  3. Persistent mapping of ID -> interval (gffutils)

array() method: signal over many intervals

  • Here, we extract the underlying signal for called peaks: signal = mapped reads; windows = called peaks
In [25]:
# Construct an array in parallel
#-----------------------------------------------------------------------------------
import pybedtools
bam = metaseq.genomic_signal('trimmed-data/G1E-ER_Gata1_chr7.bam', 'bam')
windows = pybedtools.BedTool('trimmed-data/G1E-ER_Gata1_chr7.broadPeak')
a = bam.array(windows, bins=100, processes=8, preserve_total=True)
#-----------------------------------------------------------------------------------
print "number of peaks:", len(windows)
print "array shape:", a.shape

number of peaks: 1174
array shape: (1174, 100)
  • Peaks are not the same size, so use bins=100 to scale each to 0-100% over 100 bins
  • Use 8 cores, multiprocessing.Pool.map
  • preserve_total=True means the numbers will reflect total reads per bin, rather than average reads per bin

Plot array

Choosing colormaps can be tricky. Some options:

  • normalize each row (-=mean; /=std)

  • transform the data:

    np.log1p(a)

  • Truncate colormap max:

    matplotlib.colors.Normalize(vmax=scipy.stats.scoreatpercentile(a, 99))

  • Other custom colormap:

    metaseq.colormap_adjust.smart_colormap (see docs)

Plot summary ("meta-peak")

In [28]:
def ci(a):
    "Column-wise 95% CI"
    return 1.96 * a.std(axis=0) / np.sqrt(len(a))

fig=figure(); ax=fig.add_subplot(111);
x = range(0, 100)
#--------------------------------------------------------------------------
col_mean = a.mean(axis=0)  # recall: "preserve_total=True"
ax.set_ylabel('avg reads per bin')
#--------------------------------------------------------------------------
fill_between(x, col_mean-ci(a), col_mean+ci(a), alpha=0.1, edgecolor='None')
plot(a.mean(axis=0), '0.5'); xlabel('bins');

Mini-browsers as callbacks

"mini-browsers" have a plugin-like architecture, and are designed to operate as callbacks.

Set one up, and call the plot() method with an interval

Here, we get an interval using __getitem__ on the attached gffutils database

In [41]:
from metaseq.minibrowser import GeneModelMiniBrowser
control = metaseq.genomic_signal('trimmed-data/G1E-ER_Input_chr7.bam', 'bam')
mb = GeneModelMiniBrowser(
        [bam, control],
        db=d.db,
        plotting_kwargs=[dict(color='r'), dict(color='.5')],
        )

def callback(i):
    interval = d.db[i]
    interval.start -= 1000; interval.stop += 1000
    mb.plot(interval)
    xlabel(interval.chrom); axis(xmin=interval.start, xmax=interval.stop)

# *CLICK*
callback('ENSMUSG00000030403')
